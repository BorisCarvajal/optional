    Struct Optional

    Optional type. Also known as a Maybe type in some languages.

    struct Optional(T) ;

    This can either contain a value or be empty. It works with any value, including values that can be null. I.e. null is a valid value that can be contained inside an optional if T is a pointer type (or nullable)

    It also has range like behavior. So this acts as a range that contains 1 element or is empty.

    All operations that can be performed on a T can also be performed on an Optional!T. The behavior of applying an operation on a no value or a null pointer is well defined and safe.
